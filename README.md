# Dhanlaxmi Novelty App

A Flutter-based wholesale ordering application built for Dhanlaxmi Novelty. The app allows retailers to browse products, add items to cart, and place orders efficiently.

## Features

* User Authentication (Firebase Auth)
* Product Categories
* Product Listing
* Shopping Cart
* Order Management
* Firestore Database Integration
* Responsive Flutter UI
* Admin Product Management

## Tech Stack

* Flutter
* Dart
* Firebase Authentication
* Cloud Firestore
* Cloudinary (Image Storage)
* GitHub

## Installation

1. Clone the repository

```bash
git clone https://github.com/krunalpativala/dhanlaxmi-novelty-app.git
```

2. Open project

```bash
cd dhanlaxmi-novelty-app
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
